# WebAuthn 設備使用指南

## 支援的認證方式

WebAuthn 支援兩種認證器類型：

### 1. 平台認證器（Platform Authenticator）
- **手機/平板內建的生物驗證**
  - 指紋辨識
  - 臉部辨識（Face ID、Windows Hello）
  - 其他生物識別技術
- **優點**：無需額外設備，使用方便
- **適用場景**：手機、平板、支援 Windows Hello 的電腦

### 2. 跨平台認證器（Cross-Platform Authenticator）
- **USB 安全性金鑰**（如 YubiKey、Google Titan Key）
- **NFC 安全性金鑰**
- **藍牙安全性金鑰**
- **優點**：可在多個設備間使用
- **適用場景**：需要高安全性的環境，或沒有內建生物驗證的設備

## 使用建議

### 在手機上使用（推薦）
1. 使用手機瀏覽器打開簽名頁面
2. 點擊「使用生物驗證簽名」
3. 使用手機的指紋或臉部辨識完成驗證
4. **這是最方便的方式，無需額外設備**

### 在桌面電腦上使用

#### 選項 1：使用手機（推薦）
1. 在手機瀏覽器打開相同的簽名頁面
2. 使用手機的生物驗證完成簽名
3. **優點**：無需購買額外設備

#### 選項 2：使用 USB 安全性金鑰
1. 準備一個 USB 安全性金鑰（如 YubiKey）
2. 將金鑰插入電腦的 USB 埠
3. 點擊「使用生物驗證簽名」
4. 按照提示觸摸或按下金鑰按鈕
5. **優點**：高安全性，適合企業環境

#### 選項 3：使用 Windows Hello（如果支援）
1. 如果您的電腦支援 Windows Hello
2. 可以使用指紋或臉部辨識
3. 系統會自動使用平台認證器

## 常見問題

### Q: 為什麼系統要求我插入 USB 金鑰？
**A:** 這通常發生在桌面電腦上，因為：
- 電腦沒有內建的生物驗證功能（如 Windows Hello）
- 系統檢測到已連接的 USB 安全性金鑰
- 瀏覽器偏好使用外部認證器

**解決方法：**
- 使用手機瀏覽器進行簽名（最簡單）
- 或使用 USB 安全性金鑰完成驗證

### Q: 可以在手機上使用 USB 金鑰嗎？
**A:** 可以，但需要：
- USB-C 轉接頭（Android）
- Lightning 轉接頭（iPhone）
- 或使用 NFC 安全性金鑰（如果手機支援 NFC）

### Q: 如何知道我的設備支援哪種認證方式？
**A:** 
- **手機/平板**：通常支援平台認證器（指紋/臉部）
- **桌面電腦**：
  - 支援 Windows Hello = 可使用平台認證器
  - 不支援 = 需要使用 USB 金鑰或手機

### Q: 可以同時註冊多個設備嗎？
**A:** 可以！系統支援註冊多個設備：
- 手機（指紋/臉部）
- 平板（指紋/臉部）
- USB 安全性金鑰
- 等等

每個設備都可以用來進行簽名。

## 安全建議

1. **優先使用手機生物驗證**：最方便且安全
2. **保護您的設備**：設定螢幕鎖定和生物驗證
3. **USB 金鑰保管**：如果使用 USB 金鑰，請妥善保管
4. **不要分享憑證**：每個設備的憑證都是獨特的，不要分享給他人

## 技術說明

系統會自動選擇最適合的認證器：
- 如果有平台認證器（手機生物驗證），優先使用
- 如果沒有，可以使用跨平台認證器（USB 金鑰）
- 如果都沒有，可以回退到傳統 Canvas 簽名

這樣的設計確保了最大的兼容性和用戶體驗。
